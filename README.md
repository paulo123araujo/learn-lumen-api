# Learning Lumen Framework

Developing an rest api with Lumen framework

### Requirements

 - Docker
 - Docker compose

### How to run

```
$ docker-compose up -d --build
```

### What am I using on this project?

 - Docker compose
 - PHP 7
 - Lumen
 - MySQL
 - PHPMyAdmin
